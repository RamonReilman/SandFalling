# Faling Sand cellular automata
## Author
- Ramon Reilman

## Deps
- Raylib, version 5.0

## Installation
```
git clone git@github.com:RamonReilman/SandFalling.git
cd SandFalling
mkdir build
cmake ..
make
./Sandfall
```
## Commandline
```
./Sandfall x > x = int: holds cellsize
```

## Controls
- W, speed up
- S, slowdown
- Space, start/stop
- Leftmousebtn, toggle cell state
